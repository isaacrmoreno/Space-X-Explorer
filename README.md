## Description 📝

Space X web app that utilizes [Apollo](https://www.apollographql.com/), [GraphQl](https://graphql.org/) and [Ionic](https://ionicframework.com/) framework 🔗

Original video, The Blue Pearl can be viewed [here](https://www.youtube.com/watch?v=IMVjlgWby74)
## Setup/Installation Instructions 📁

### Prerequisites 🖥️

- [Node](https://nodejs.org/en/)
- A text editor like [VSCode](https://code.visualstudio.com/)

1. Clone the repository: `% git clone https://github.com/isaacrmoreno/SpaceX.git`
2. Navigate to the `SpaceX` directory on your computer
3. Open your preferred text editor to view code
4. To start a development server and view project in the browser:
   - Navigate to `SpaceX in your command line
   - Run command `yarn install` to install dependencies
   - Optional: Run command `ionic build` to bundle files
   - Finally, run the command `ionic serve` to start a development server
